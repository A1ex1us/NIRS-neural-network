# NIRS-neural-network
Научно исследовательская работа - нейросеть по решению задачи о выпуклости многоугольника

---

## Описание проекта

Данный проект реализует два подхода для определения выпуклости многоугольников на изображениях:
1. **Традиционный метод** - с использованием компьютерного зрения и геометрических вычислений
2. **Нейросетевой подход** - с созданием и обучением нейронной сети с нуля

Проект был разработан в рамках научно-исследовательской работы (НИРС).

---

## Цели и задачи

### Цель работы:
Определить, является ли многоугольник на исходном изображении выпуклым, традиционным методом и с помощью нейронной сети.

### Задачи:
1. Решить задачу аналитическим методом (традиционным)
2. Изучить основные концепции нейронных сетей
3. Создать dataset для обучения нейронной сети
4. Реализовать нейросеть на Python и проверить ее обучаемость
5. Протестировать нейросеть на изображениях многоугольников
6. Реализовать разделение невыпуклых многоугольников на выпуклые для стандартного метода

---

## Структура проекта

### Основные файлы:

- **main.py** - основной файл для обучения нейронной сети
- **nirs_test.py** - реализация традиционного метода определения выпуклости
- **polygon_generator.py** - генератор случайных многоугольников
- **zipper.py** - создание датасета из сгенерированных изображений
- **utils.py** - вспомогательные функции для загрузки данных
- **test.py** - тестирование обученной нейронной сети

### Файлы данных:

- **.npy файлы** - сохраненные веса и смещения обученной нейросети
  - `weights_input_to_hidden.npy` - веса между входным и скрытым слоем
  - `weights_hidden_to_output.npy` - веса между скрытым и выходным слоем
  - `bias_input_to_hidden.npy` - смещения для скрытого слоя
  - `bias_hidden_to_output.npy` - смещения для выходного слоя

- **.npz файлы** - датасеты для обучения
  - `network_dataset.npz` - основной датасет с изображениями и метками
